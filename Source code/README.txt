TUTORIAL


Щоб встановити локалізацію, вам потрібно пройтися за цими пунктами:
1. Відкрити Steam -> Бібліотека -> Керування -> Подивитися локальні файли
2. Відкрити папку Translation
3. Копіювати зверху шлях до папки
4. У програмі натиснути «Завантажити»
5. Вставити в рядок вище шлях до папки Translation і натиснути «Вибрати папку».
6. Очікувати скачування.

Готово! Ви можете в налаштуваннях гри поставити українську мову.
Програму ви можете собі залишити, щоб у наступні рази оновлювати локалізацію.

Також у папці Документи/UKDownloader_cash зберігається кеш інсталяції, можете після встановлення локалізації очищати цю папку.
